# BOB LAND - GROUPE 1 - 2023/2024

Bob Land a Game centred around a visual and interactive simulation of natural selection created by 
Morad, Aymen, Yassmine, Oumaima, Maxime et Yassine

## Information sur les touches :
Dans le jeu :
'O' : pour ouvrir les informations des autres touches
'T' et 'G' : pour augmenter et diminuer la vitesse de déplacement
'Y' et 'H' : pour augmenter et diminuer la vitesse du zoom
Flèche du haut/bas : pour zoomer et dezommer
Curseur de la souris sur les bords de l'écran pour se déplacer dans la map
